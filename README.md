# HR AGENT AI - Корпоративный портал

Современное веб-приложение для HR-агента с AI-чатом, форумом и административной панелью.

## Возможности

### AI-Чат
- 💬 Интерактивный чат с AI-ассистентом для HR-вопросов
- 📝 Множественные чаты с возможностью редактирования названий
- 💾 Автоматическое сохранение истории чатов в localStorage
- 🔄 Переключение между чатами

### Форум
- 📋 Категории форума (Адаптация, Общение, Вопросы HR, Обучение, Офис)
- 📰 Просмотр тем и статей
- 💬 Система комментариев и ответов
- 👍 Лайки для тем и комментариев
- 🔍 Поиск по форуму
- 📌 Закрепленные темы

### Административная панель
- 📄 Управление документами (загрузка, просмотр, удаление)
- 📚 Управление базой знаний
- 🔐 Доступ только для администраторов
- 📊 Отображение статуса загрузки файлов

### Общие возможности
- 🔐 Система аутентификации (регистрация/вход)
- 👤 Профиль сотрудника с детальной информацией
- 🌓 Темная и светлая тема
- 📱 Адаптивный дизайн для мобильных устройств
- 🎨 Современный UI с использованием шрифта Rubik
- 📹 Быстрый доступ к видеозвонкам с HR

## Технологии

- **Frontend**: React 19.2.5
- **Build Tool**: Vite 8.0.10
- **Styling**: CSS3 с кастомными переменными
- **Icons**: Lucide React
- **Backend API**: REST API (документация в API_DOCUMENTATION.md)

## Установка

```bash
npm install
```

Создайте файл `.env` в корне проекта:

## Основные компоненты

### App.jsx
Главный компонент приложения, управляющий:
- Аутентификацией пользователя
- Навигацией между разделами (чат, форум, админ-панель)
- Состоянием темы
- Управлением чатами

### Auth.jsx
Компонент аутентификации с формами входа и регистрации.

### ChatMain.jsx
Интерфейс чата с AI-ассистентом, поддержка множественных чатов.

### ForumHome.jsx / ForumList.jsx / TopicView.jsx / ArticleView.jsx
Компоненты форума для отображения категорий, списка тем, просмотра тем и статей.

### AdminPanel.jsx
Административная панель для управления документами и базой знаний.

## API Integration

Приложение интегрируется с backend API. Полная документация доступна в файле `API_DOCUMENTATION.md`.

Основные эндпоинты:
- `/auth/register` - Регистрация
- `/auth/login` - Вход
- `/forum/categories` - Категории форума
- `/forum/topics` - Темы форума
- `/forum/topics/:id/replies` - Ответы к теме
- `/admin/documents` - Управление документами
- `/admin/knowledge-base` - Управление базой знаний

## Роли пользователей

- **employee** - Обычный сотрудник (доступ к чату и форуму)
- **admin** - Администратор (полный доступ, включая админ-панель)

## Особенности интерфейса

### Боковая панель
- Переключение между чатом и форумом
- История чатов / последние темы форума
- Профиль пользователя
- Кнопка видеозвонка
- Доступ к админ-панели (для администраторов)

### Главная панель
- **Режим чата**: Интерфейс чата с AI, редактирование названий чатов
- **Режим форума**: Просмотр категорий, тем, статей
- **Админ-панель**: Управление документами и базой знаний

### Профиль
Детальная информация о сотруднике:
- Идентификация (ID, email)
- Личные данные (ФИО, дата рождения)
- Рабочие данные (должность, отдел, дата найма)
- Контакты (телефон, email)

### Темы
- Автоматическое переключение между светлой и темной темой
- Сохранение предпочтений в localStorage
