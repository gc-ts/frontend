# HR AGENT AI

Современный AI-чат для HR-агента с поддержкой темной и светлой темы.

## Возможности

- 💬 Интерактивный чат с AI-ассистентом
- 🌓 Темная и светлая тема
- 📱 Адаптивный дизайн для мобильных устройств
- 💼 Профиль сотрудника с детальной информацией
- 📝 Редактирование названий чатов
- 🎨 Современный UI с использованием шрифта Rubik
- 📹 Быстрый доступ к видеозвонкам с HR

## Технологии

- React 18
- Vite
- CSS3 с кастомными переменными
- Lucide Icons

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Особенности интерфейса

- **Боковая панель**: История чатов, профиль пользователя, кнопка видеозвонка
- **Главная панель**: Чат с AI-ассистентом, редактирование названий
- **Профиль**: Детальная информация о сотруднике (идентификация, личные данные, рабочие данные, контакты)
- **Темы**: Автоматическое переключение между светлой и темной темой
